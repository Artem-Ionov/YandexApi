Чтобы запустить этот проект локально на компьютере выполните следующее:

1. Настройте среду разработки python (версия указана в файле runtime.txt) и установите django (версия указана в файле requirments.txt).
2. Выполните в терминале следующие команды: python manage.py makemigrations; python manage.py migrate; python manage.py createsuperuser; python manage.py runserver.
3. Откройте браузер и перейдите по адресу http://127.0.0.1:8000/?public_link=, где после public_link= укажите публичную ссылку на папку Яндекс диска.
